## 贪心算法

**定义:** Iteratively make"myopic"decisions, hope everything
works out at the end.

**举例:** Dijkstra最短路径

**算法设计:** 无通用范式

#### 对比分治算法

1. 对多个问题易提出多种贪心算法
2. 容易分析时间复杂度(对比主定理)
3. 难分析算法正确性(与简单归纳证明相比) 

**注意:** 大多数贪心算法都不正确(不要盲目相信你的直觉)

#### 算法正确性

1. Induction. ("greedy stays ahead")
2. Exchange argument(交换顺序证明)
3. Whatever works!
